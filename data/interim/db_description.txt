
==================================================

1. ROOT OBJECT STRUCTURE

{
poi_id: string
type: string
sub_type: string | null
name: string
address: string
gu_name: string
image: string | null
location: object
description: string
attributes: object
stats: object
}

==================================================
2. FIELD SPECIFICATION
2.1 poi_id

Type: string

Description: POI 고유 식별자

Example:

"2456536"

"poi_832950"

2.2 type

Type: string

Allowed Values:

attraction

cafe

restaurant

Description: POI의 대분류 타입

2.3 sub_type

Type: string | null

Description: 세부 카테고리

Example:

"관광단지"

"사찰"

"카페/커피숍"

2.4 name

Type: string

Description: 장소명

2.5 address

Type: string

Description: 전체 주소 문자열

2.6 gu_name

Type: string

Description: 행정구 이름 (예: 강남구)

Note: 일부 POI에만 존재

2.7 image

Type: string | null

Description: 대표 이미지 경로

Storage Policy:

관광지(attraction): 외부 URL
예) http://tong.visitkorea.or.kr/
...

카페/식당: 로컬 이미지 파일명
예) "832950.png"

이미지가 없는 경우 null

2.8 location

{
lat: number
lng: number
}

lat: 위도 (float)

lng: 경도 (float)

2.9 description

Type: string

Description: 장소에 대한 상세 설명 텍스트

==================================================
3. ATTRIBUTES (SEMANTIC INFORMATION)

의미 기반 정보 및 추천/분석용 확장 필드

{
duration: string
activity: object
photospot: boolean
indoor_outdoor: string
keywords: string[]
summary: string
themes: string[]
mood: string[]
visitor_type: string[]
best_time: string[]
parking: true (optional)
}

3.1 duration

Type: string

Example:

"1~2시간"

"2~3시간"

3.2 activity

{
label: string
level: number
}

활동 강도 정보

3.3 photospot

Type: boolean

Description: 사진 촬영 명소 여부

3.4 indoor_outdoor

Type: string

Example:

실내

실외

복합

3.5 keywords

Type: string[]

Description: 핵심 키워드 목록

3.6 summary

Type: string

Description: 한 문장 요약 (AI 생성)

3.7 themes

Type: string[]

Description: 장소 테마 정보

3.8 mood

Type: string[]

Description: 장소 분위기

3.9 visitor_type

Type: string[]

Description: 추천 방문자 유형

3.10 best_time

Type: string[]

Description: 추천 방문 시간대

3.11 parking

Type: boolean (true only)

Description:

주차 가능한 경우에만 필드 존재

false / null 값은 저장하지 않음

==================================================
4. STATS (STATISTICS)

{
views: number | null
likes: number | null
bookmarks: number | null
rating: number | null
review_count: number | null
}

외부 플랫폼 또는 사용자 반응 기반 통계

데이터가 없는 경우 null
